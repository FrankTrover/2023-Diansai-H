**背景说明**：
你是一个资深的嵌入式软件及数字信号处理（DSP）工程师。请为“2026 生物医学工程学院电子设计大赛”的“信号分离装置”编写核心逻辑代码框架（如 `main.c` 及相关算法处理模块）。该代码将运行在系统的主控芯片上，负责高速采集混合信号、执行识别算法并控制外部 DDS 芯片输出分离信号。

**硬件架构约束**：
1. **输入采集**：前端为增益为 1 的独立加法器电路（仅通过信号线提供混合信号 C，主控无法与其通信）。ADC 采用 **AD7616**（16位，最高 1MSPS），你需要配置高速数据采集，以满足至少 500kSPS 的采样率。
2. **输出生成**：使用两片独立的 DDS 芯片（**AD9833** 和 **AD9834**）通过 SPI 接口同时输出分离后的信号 A' 和 B'。
3. **状态显示**：使用 LCD 屏幕进行 UI 展示。无按键设计，上电即全自动运行。

**业务流程与时间限制**：
* 系统上电完成初始化后，直接进入“采集 -> 分析 -> 识别 -> 输出 -> 刷新屏幕”的全自动循环。
* 单次信号分离的完整处理耗时必须**严格小于 20s**。

**核心算法与分离需求（至关重要）**：
* 混合信号 C = A + B。频率范围 20kHz ~ 100kHz，且 fA < fB。输入信号峰峰值均为 1V，直流偏置不详（ad7616范围内）。
* **场景自适应**：算法需能自动识别并分离以下三种情况：
  1. fA=50kHz, fB=100kHz 的双正弦波。
  2. 频率为 10kHz 整数倍的双正弦波。
  3. 信号A和B分别为正弦波或三角波，频率分别为5kHz的整数倍。要求装置能正确分离出信号A‘和B’


**信号输出与显示质量要求**：
* **稳定不漂移**：分离后输出的 A' 和 B' 波形需无失真。你计算出的频率控制字（FTW）必须极其精准，确保输出信号与原信号能保持同频，在示波器上连续稳定显示且不漂移（需考虑频率测量分辨率的优化）。
* **LCD 界面**：实时对比显示输入端信息（检测推算出的原始信号 A 和 B 的频率值及波形种类）与输出端信息（下发给两片 DDS 的实际频率值和波形状态）。
